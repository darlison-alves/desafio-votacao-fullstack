## Executa API

```shell
  ./mvnw spring-boot:run
```

Para as dependencias de banco e redis, estão sendo usando serviços externos


## Executar Front

```shell
    yarn install ou npm install
```

Apos instalação das dependencias, rodar comando para subir o front 

```shell
    yarn dev ou npm run dev
```

A aplicação ja esta apontando para api na porta 8080
